int32 FOnlineSessionInfoAccelByteV1::GetTeamIndex ( const FUniqueNetId &  UserId) const

Attempts to get a team index for a user, returns INDEX_NONE if not found.

Note that team indices are just mapped from 0 to how ever many teams are matched. If you have special team numbers you'll want to map these values on the client side.
